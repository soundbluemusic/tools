# 🎵 메트로놈 (Metronome)

음악가를 위한 정밀 메트로놈 앱입니다.

## 경로

`/metronome`

## 기능

### BPM 조절
- 범위: 20-300 BPM
- 슬라이더 또는 직접 입력으로 조절
- +/- 버튼으로 미세 조정

### 박자 설정
- 지원 박자: 2/4, 3/4, 4/4, 5/4, 6/8, 7/8, 9/8, 12/8
- 첫 박 강조 (악센트)

### 탭 템포
- 화면을 탭하여 템포 감지
- 최근 탭 기록 기반 BPM 자동 계산

### 시각적 피드백
- 현재 박자 시각적 표시
- 비트에 맞춘 애니메이션

## 기술 구현

- **오디오**: Web Audio API 기반 정밀 타이밍
- **스케줄링**: 오디오 컨텍스트의 `currentTime` 활용으로 정확한 비트 재생
- **상태 관리**: React 상태 + localStorage 영속화

## 키보드 단축키

| 키 | 동작 |
|:---|:-----|
| `Space` | 재생/정지 |
| `↑` / `↓` | BPM 증가/감소 |
