# 🥁 드럼머신 (Drum Machine)

드럼 패턴 연습용 시퀀서 앱입니다.

## 경로

`/drum`

## 기능

### 패턴 시퀀서
- 16스텝 시퀀서 그리드
- 다양한 드럼 사운드 지원 (킥, 스네어, 하이햇, 탐, 심벌 등)
- 클릭으로 스텝 토글

### 재생 컨트롤
- 템포 조절 (BPM)
- 루프 재생
- 루프 길이 및 횟수 설정

### MIDI 지원
- MIDI 파일 가져오기
- MIDI 파일 내보내기
- 드럼 패턴 저장/불러오기

### 사운드 합성
- Web Audio API 기반 실시간 합성
- 드럼 사운드 합성기와 동일한 사운드 엔진

## 드럼 사운드 목록

| 이름 | 설명 |
|:-----|:-----|
| Kick | 베이스 드럼 |
| Snare | 스네어 드럼 |
| Hi-Hat (Closed) | 닫힌 하이햇 |
| Hi-Hat (Open) | 열린 하이햇 |
| Clap | 핸드 클랩 |
| Tom (High/Mid/Low) | 탐탐 |
| Crash | 크래시 심벌 |
| Ride | 라이드 심벌 |

## 기술 구현

- **오디오**: Web Audio API 기반 합성
- **시퀀싱**: 정밀한 오디오 스케줄링
- **MIDI**: Web MIDI API 및 파일 파싱

## 키보드 단축키

| 키 | 동작 |
|:---|:-----|
| `Space` | 재생/정지 |
| `1-9` | 해당 트랙 음소거 토글 |
