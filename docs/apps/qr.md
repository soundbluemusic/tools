# 📱 QR 코드 생성기 (QR Code Generator)

투명 배경의 고해상도 QR 코드를 생성하는 앱입니다.

## 경로

`/qr`

## 기능

### QR 코드 생성
- 텍스트 또는 URL 입력
- 실시간 미리보기
- 다양한 크기 옵션

### 커스터마이징
- 투명 배경 옵션
- 전경색 선택
- 배경색 선택 (투명이 아닌 경우)
- 에러 교정 레벨 (L/M/Q/H)

### 내보내기
- PNG 포맷 다운로드
- 다양한 해상도 선택 (128px ~ 1024px)

## 사용 예시

### URL 공유
```
https://example.com
```

### 연락처 (vCard)
```
BEGIN:VCARD
VERSION:3.0
N:홍길동
TEL:010-1234-5678
END:VCARD
```

### Wi-Fi 연결
```
WIFI:T:WPA;S:네트워크이름;P:비밀번호;;
```

## 에러 교정 레벨

| 레벨 | 복원 가능 비율 | 용도 |
|:-----|:--------------|:-----|
| L (Low) | ~7% | 깨끗한 환경 |
| M (Medium) | ~15% | 일반적 사용 |
| Q (Quartile) | ~25% | 약간의 손상 예상 |
| H (High) | ~30% | 로고 삽입, 손상 많은 환경 |

## 기술 구현

- **라이브러리**: QRious
- **렌더링**: Canvas API
- **다운로드**: Blob + URL.createObjectURL
