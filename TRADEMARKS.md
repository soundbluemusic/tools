# 상표 가이드라인 (Trademark Guidelines)

## 상표

"SoundBlueMusic" 이름과 로고는 SoundBlueMusic의 상표입니다.

## 로고 색상 (Logo Color)

| 속성 | 값 |
| --- | --- |
| 색상명 | Medium Purple (미디엄 퍼플) |
| HEX | `#9370DB` |
| RGB | `rgb(147, 112, 219)` |
| HSL | `hsl(260, 60%, 65%)` |

### 색상 선택 이유

**창의성과 상상력**을 상징하는 보라색을 선택했습니다. 보라색은 역사적으로 예술적 표현, 혁신, 창의적 사고와 연관되어 왔습니다. 음악가, 개발자, 창작자에게 영감을 주기 위해 설계된 우리 도구들의 창의적인 본질을 담아내고자 이 색상을 선택했습니다.

이 프로젝트를 포크하여 본인의 서비스로 배포할 경우:

자세한 내용은 [README.md](./README.md)의 "포크 & 배포 가이드"를 참조하세요.
