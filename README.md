# Productivity Tools

React + TypeScript + Vite 기반의 유틸리티 앱 대시보드

## 앱 목록

| 앱 | 설명 | 경로 |
|:---|:-----|:-----|
| 🎵 메트로놈 | Metronome | `/metronome` |
| 📱 QR 코드 생성기 | QR Code Generator | `/qr` |

## 시작하기

```bash
# 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 전체 검증 (타입체크 + 린트 + 테스트)
npm run validate
```

## 기술 스택

| 분류 | 기술 |
|:-----|:-----|
| 프레임워크 | React 18 |
| 라우팅 | React Router 7 |
| 언어 | TypeScript 5 |
| 빌드 | Vite 5 |
| 테스트 | Vitest + React Testing Library |
| 배포 | Cloudflare Pages |
